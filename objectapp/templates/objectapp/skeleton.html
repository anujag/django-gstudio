Welcome!

This application is an attempt to collaborative framework for online construction and publication of knowledge networks and ontologies.

Currently gnowledge studio provides support for Objecttypes and GBObjects. Using objecttypes you can define different kinds of concepts, like "concept names", "action names", "scientific concept", "phylem" etc. GBObjects are any regular classes, which can be defined as members of Objecttypes. Both gbobjects and objecttypes can have parent, siblings and children to create hierarchical structures. In our next release we will have objects as members of gbobjects. While we shape this project, if you are interested in joining, visit us at metastudio.org.
